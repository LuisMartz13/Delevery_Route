# System Brief

## Nombre del sistema
DeliveryRoute API

## Descripción
DeliveryRoute API es un sistema que permite registrar pedidos y dar seguimiento a su estado de entrega.

## Problema que resuelve
Muchos negocios necesitan controlar sus pedidos y saber en qué estado se encuentran (pendiente, en proceso o entregado). Este sistema permite gestionar pedidos de forma simple mediante una API.

## Usuarios del sistema
- Administrador
- Repartidor
- Cliente (indirectamente)

## Flujo principal del MVP
1. Crear pedido
2. Consultar pedido
3. Cambiar estado del pedido
4. Listar pedidos

El sistema permite tener control básico del proceso de entrega.
